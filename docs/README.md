## 01-战斗机制.md 章节地图

战斗机制是单一大文档，涵盖以下主题，每个主题对应 `src/Xjdl.Core` 的一个模块（技术栈见 [ADR 0001](adr/0001-引擎与核心语言选型.md)：C#/.NET 核心 + Godot 表现层）：

| 文档章节 | 代码模块 |
| -------- | -------- |
| 地图与格子（六角格、轴向坐标、相邻/距离） | `Core/Hex` |
| 战斗流程 · 三张表（常规进攻/对攻/遭遇战）、结果代码 | `Core/Combat` |
| 战斗韧性与战损结算（线性衰减、整除约束） | `Core/Combat` |
| WEGO 回合结算顺序（阶段 0–9、快照、确定性裁定） | `Core/Turn` |
| 火力支援、堆叠、控制区、机动冲突、撤退/推进裁定 | `Core/Turn` + `Core/Combat` |
| 作战学说与数值微调（预算、签名机制、克制） | `Core/Doctrine` + `Data/Doctrines` |
| 昼夜机制（视野/射程/火力比/机动/控制区修正、夜战标志） | `Core/Turn` + `Core/Modifiers` |
| 战争迷雾（三级可见度、刷新时点、配置项） | `Core/Fog` |
| 地形系统（移动消耗、防御 DRM） | `Core/Terrain` + `Data/Terrain` |
| 技能卡系统（CP 经济、规模档位、时机标签、样卡） | `Core/Cards` + `Data/Cards` |
